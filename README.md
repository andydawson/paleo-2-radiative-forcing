# paleo-2-radiative-forcing



1. NeotomaDownloads.R ------------------ Downloads from the Neotoma Database
2. Mutate Data.R ---------------- Makes the data into a more usable format, long, wide, and extra long and extra wide
3. Moddeling.R -------------- Runs models of the albedo with the data and calculates RF. The bottom is where we left off.
4. Soon to be Visualization.R ------------ Makes all of the pretty viuals

Required files
 - blue_sky_monthly_2000-2009.tif
 - taxaNames_classified.csv
